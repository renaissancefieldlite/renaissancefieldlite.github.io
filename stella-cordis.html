Package compatibility

Mac and PC packages are ready.

Mac

AU + VST3 package

  • macOS 12 Monterey or newer recommended
  • Apple Silicon or Intel Mac
  • Ableton Live 10/11/12, Logic Pro, or another AU/VST3 host
  • Install paths: Components and VST3 plugin folders

PC

Windows VST3 package

  • Windows 10 or Windows 11, 64-bit
  • Intel or AMD x64 processor
  • Ableton Live 10/11/12 or another VST3 host
  • Microsoft Visual C++ Redistributable 2015-2022 may be required

Host

Recommended session specs

  • 8 GB RAM minimum, 16 GB recommended
  • Modern multi-core CPU for polyphonic pads and mirror chords
  • 44.1 kHz or 48 kHz host sessions
  • Raise buffer size for dense chords and long releases
